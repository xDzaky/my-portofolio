<Prose>
  <p>
    Project ini saya buat sebagai latihan mendesain landing page yang ringan namun tetap dinamis. Fokus
    utama ada pada bagaimana menu hamburger bisa menyatu dengan konten dan tetap mudah dioperasikan
    pada layar kecil.
  </p>
  <p>
    Saya menyiapkan struktur komponen reusable, kemudian mengatur animasi transisi dengan CSS supaya
    menu terasa responsif dan tidak mengganggu fokus pengguna. Percobaan ini menjadi dasar saya untuk
    memahami micro-interaction sebelum melanjutkan ke framework JavaScript modern.
  </p>
  <p>
    Walaupun masih dalam tahap konsep, project ini terus saya iterasi untuk menguji ide tata letak dan
    pattern navigasi yang ramah pengguna.
  </p>
</Prose>
