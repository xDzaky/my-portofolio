<Prose>
  <p>
    Proyek ini berawal dari kebutuhan sederhana: memastikan inventaris laboratorium sekolah selalu
    terpantau tanpa harus membuka tumpukan catatan manual. Saya mulai dengan merancang struktur data
    stok, menyiapkan alur input, dan mendesain tampilan dashboard yang mudah dibaca oleh teman satu
    tim maupun guru pembimbing.
  </p>
  <p>
    Solusinya berupa aplikasi web statis dengan bantuan JavaScript dan Python scripting untuk
    sinkronisasi data. Antarmuka menyediakan form stok masuk/keluar, ringkasan kuantitas, serta
    riwayat perubahan. Saya juga menyiapkan integrasi awal ke Google Spreadsheet sebelum rencana
    migrasi ke database penuh diterapkan.
  </p>
  <p>
    Hasilnya, proses pencatatan inventaris menjadi lebih konsisten dan dapat dipantau kapan saja.
    Pekerjaan administratif mingguan berkurang karena data sudah tersusun rapi dan bisa dipakai ulang
    untuk laporan sekolah.
  </p>
</Prose>
